# Observacionismo for Local-First AI Agents

This is a public-safe map of the current Lutren / MEDIOEVO open-source work on
evidence-gated AI agents.

The practical thesis is narrow:

AI agents that can use tools should leave evidence before action, declare
uncertainty, pause on risky operations, and produce witness logs that another
human or agent can replay.

This does not claim to prove new physics, consciousness, medical safety, social
prediction, or autonomous agent safety. It is an engineering control layer for
local-first agent workflows.

## Current Signal

The useful public question is not whether an AI agent sounds confident.

The useful question is whether the agent can:

- show what evidence it used;
- label certainty, inference, and unknowns;
- survive negative controls;
- stop before destructive or external actions;
- leave a witness log;
- hand off context to another agent without losing the thread.

That is the public lane of Observacionismo: evidence before action.
